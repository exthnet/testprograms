# OpenMP使用時のFortranポインタ宣言時の注意の話

* 具体的な実例：fomp3.f90
	* func1は宣言時null代入なし
		* ポインタpがローカル変数の挙動を示す
	* func2は宣言時null代入あり
		* ポインタpが共有変数の挙動を示す

## 実行例

> $ !./
> ./fomp3
>  tid           0
>  tid           1
>  func1           0   1.00000000000000
>  func1           1   2.00000000000000
>  tid           0
>  tid           1
>  func2           0   2.00000000000000
>  func2           1   2.00000000000000
